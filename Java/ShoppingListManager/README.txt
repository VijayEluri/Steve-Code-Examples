This README file explains how to use the ShoppingListManager GUI to read and write data to files.  The file read/write funtionality is part of the ShoppingListManager application and is accessed by clicking either the "Read Items" or "Write Items" buttons.

# Assumtions

The ShoppingListManager GUI must be running.

# Write Data
1) Add some shopping list items and quantities to the GUI
2) Click the "Write Items" button
3) Select a file path to write data to.  For example, you might choose "C:\data".
4) Click the "Write" button.

# Read Data
1) You may choose to have an empty shopping list, it doesn't matter
2) Click the "Read Items" button
3) Find and select the file that contains data that you previously wrote
4) Click the "Read" button.
